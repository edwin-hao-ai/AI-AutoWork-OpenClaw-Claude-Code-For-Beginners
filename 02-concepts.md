## 一、龙虾（OpenClaw）是什么？它为什么是操作系统？ 约 6 分钟

**大白话类比：**

如果把你的电脑比作一个工厂，早期的 AI（比如 ChatGPT、豆包）只是坐在顾问室里的理论专家，摸不到你的机器。它只会解答，但**没有手**。

龙虾（OpenClaw）是真正的**下一代 AI 操作系统**。它相当于给 AI 装上了真实的手和脚，可以直接接管你的电脑环境，直接去开文件、跑环境、控制浏览器。

**整套系统的流水线：**
你（说大白话） 龙虾（听懂 + 想方案 + 写代码 + 操作电脑） 结果直接给你。没动一次鼠标。

## 二、玩转新时代的三大武器  约 9 分钟

> [PPT参考] 第5张幻灯片：三大武器图解：大脑（Claude Code）+ 双手（龙虾）+ 保险柜（Git）

要驾驭这个强大的操作系统，你不需要学编程，只需要掌握**三大核心武器的合理使用**。它们是怎么分工的？

### 武器 1：Claude Code  你的超级大脑 / 顶级程序员
当我们遇到一个问题，比如把这100个Excel里的销售额加起来。Claude Code 就是那个想出解法并快速敲键盘产出代码的大脑。它负责规划逻辑、编写代码、分析报错。

### 武器 2：OpenClaw（龙虾）  你的执行双手 / 底层操作系统
光有代码没用，你得能运行。龙虾负责落地执行。Claude Code 写完代码后，由龙虾来创造沙盒环境、运行脚本、抓取网页。
**一句话：没有龙虾，大脑的代码就只是纯文本；有了龙虾，代码就变成了自动化的真实动作。**

### 武器 3：Git  你的规矩保险柜 / 记忆库
一提到 Git，小白就觉得是程序员专属。今天，我们要把 Git **降维打击**，普通大众把它当保险柜用！
- **对于个人（本地文件版本管理）**：你写好的 AI 规则（比如怎么写小红书文案），放在里面，防篡改、就算改错了一键就能回溯。
- **对于团队（团队协作版本管理）**：你配置好了一套全自动化处理报表的流程，一键提交（Push），全团队的老板同事一拉取（Pull），大家的 AI 助手瞬间就能干一样的活。

**三大武器的梦幻连招：**
你下达需求  Git (保险柜) 自动提供底层规矩  Claude Code (大脑) 产出方案代码  龙虾 (双手) 动手干活。
