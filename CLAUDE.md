# 项目规则

## 语言要求

- 所有回复必须使用中文，包括思考过程、分析、解释和代码注释。
- 无论用户使用何种语言提问，始终以中文回答。

## 自动提交与推送

- 每次完成文件修改后，自动执行 git add、git commit 并 git push 到远程仓库。
- 提交信息使用中文，简洁描述本次修改内容。
- 无需询问用户是否提交，直接执行。

## 权限与自主操作

- 拥有最大操作权限，无需每步询问用户确认。
- 可自主执行文件读写、命令运行、git 操作等所有任务。
- 遇到明显可执行的任务直接完成，不反复确认。
- **重要限制**：不要随便删除任何文件，执行删除操作前必须先询问并获得我的确认。
