## 模块 1：怎样像用网盘一样用好 Git（保险柜）？ 约 10 分钟

> [PPT参考] 第9张幻灯片：Git 降维打击图解：本地保险柜 + 云端共享

别害怕命令行，教大家极简工作流（其实 VS Code 编辑器里点点鼠标就能做）：

**第一步：本地存档。**你想好了一个做表规则，写进了 AGENTS.md。这时候点击初始化仓库，写上一句新建了做表规矩，点击提交 (Commit)。这就等于把文件存进了本地无敌保险柜。明天你不小心删了它，一秒钟就能退回去。

**第二步：云端发给全公司。**只需点击同步 (Push)，这些规矩文件就上了 GitHub 云端。（具体 Github 注册流程会后看我提供的图文手册，像注册微信一样简单）。其他同事一点拉取 (Pull)，整个公司的 AI 标准瞬间拉齐。

## 模块 2：简单工作流 vs 复杂工作流（Skill）的合理调度 约 15 分钟

我们要根据任务难度，**合理调度**我们的大脑和双手。

**第一类：简单工作流（单兵作战）**
比如需求：帮我把桌面 0820 文件夹里所有带数字前缀的图片更名为日期格式。
这种就是单点、一次性的操作，不用定复杂的长期规矩。直接在对话框发布指令，由 Claude Code 写个几行小脚本，龙虾自动执行一下，一分钟搞定。用完即毁。

**第二类：复杂工作流与 Skill（技能）调用**
比如需求：每天定时监控某个飞书群里的消息，只要产品经理发了长段反馈，就提取出核心痛点，生成表格数据并提醒我。
对于这种需求，如果纯靠大模型现场写代码解决，就会很不稳定。这里就要引入**系统内的 Skill（技能）概念**。

你可以把 Skill 理解为已经固化好的武器配件（比如飞书消息读取 Skill、飞书发消息 Skill）。
AI 通过读取文档，能知道自己系统里配了哪些 Skill。然后自动拆解任务：第一步调飞书 Skill 监控；第二步调提取分析；第三步调发送 Skill 汇报。
**这就是我们在开头说的，系统能自动调起合适的插件来做工程组合流，稳定且强悍。**

## 模块 3：保姆级上手指南与万能大白话提示词模板（开箱即用）

> [PPT参考] 第10张幻灯片：新人的第一步安装与白话模板

很多同学问：听懂了，但我下课后怎么把环境弄跑起来？ 别慌，跟着这里的真实链接和提示词，一步步来。

### 1. 注册你的云端保险柜（GitHub）
不用懂代码，把它当成全网最安全的免费规矩云盘。
- **官方注册真实链接**：[https://github.com/](https://github.com/) 
- **小白操作**：点进去右上角的 "Sign up"，直接用邮箱注册，记住账号密码。以后全公司的 AI 流程同步都依靠它。

### 2. 安装你的超级大脑：Claude Code
前提是你的电脑装好了Node（开发大基建），只需要一行指令，大脑就位。
- **Anthropic官方真实介绍地址**：[https://docs.anthropic.com/en/docs/agents-and-tools/claude-code/overview](https://docs.anthropic.com/en/docs/agents-and-tools/claude-code/overview)
- **一键安装真正口令**：
  打开系统自带的终端（Terminal / CMD），复制粘贴这行代码并回车，等待进度条走完：
  `npm install -g @anthropic-ai/claude-code`

### 3. 安装你的执行双手：OpenClaw（全自动安装提示词）
作为小白，不懂克隆代码、不懂配环境怎么办？没关系，既然大脑（Claude Code）已经装好了，我们直接**用魔法打败魔法让大脑自己去安装这双手！**
你在终端里唤醒 Claude Code 后，直接输入下面这段大白话提示词：

> **你好，请帮我在当前目录下部署起 OpenClaw。我是一个零基础的新手，请你先检查一下我电脑的系统环境。然后你自己去 Github 上找到并克隆 OpenClaw 的官方代码仓库，自己安装它需要的 Python 环境和所有依赖包。如果中间遇到了任何报错、网络问题或依赖冲突，不要停下来问我，请你自己分析并解决报错，直到看到 OpenClaw 完整成功运行起来为止。**

*(讲师提醒：这就是 AI 操作系统的魅力，连安装它自己，它都能全自动办到！)*

### 4. 万能大白话提示词填空模板（新手日常办公发任务专用）

不要再记那些复杂的编程结构了，我们要像老板发号施令一样。以后不管你遇到了什么办公杂活，直接套用下面这个【填空题】模板：

> **【角色与目标】**
> 你现在是我的**[填写：资深数据分析师 / 飞书社群管家 / 新媒体运营总监]**。
> 我现在需要你帮我搞定这件麻烦事：**[填写：帮我把电脑桌面周报文件夹里的所有Excel合并，查出这周谁的业绩最高]**。
> 
> **【做事规矩（Plan 后 Auto）】**
> 1. 作为我的得力助手，请你不要一上来就盲干凑字数或者乱跑一通代码。
> 2. 请先给我列出一个清晰明确的 123 步执行计划（Plan）。
> 3. 请重点注意：**[附加要求填写：最终汇总表的表头一定要标成黄色，超过10万业绩的名字要标红 / 如果是文案，每段必须加 Emoji]**。
> 4. 等我看了你的计划，回复你OK之后，你再去调用你的底层系统进行自动化实施（Auto）。
> 
> **【如何交付】**
> 活儿干完了之后，不要只给我看屏幕代码，请直接把最终的文件保存到**[填写：我的桌面上]**，并通知我查收。
