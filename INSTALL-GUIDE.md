哈喽，新手魔法师！欢迎踏上 AI 自动化编程的奇妙之旅！在这篇指南里，我们会为你准备好几件强大的神器，让写代码就像点外卖一样简单。看不懂长串的代码和黑字？没关系！直接把底下的【提示词】复制给你的 AI 助手，它会亲自乖乖帮你干活！

##  第一步：认识我们的魔法包（这些都是啥？为什么要装？）

在开始念一键咒语之前，我们先来检查一下行囊里的宝贝：

### 1.  **VSCode (Visual Studio Code)**
* **这是啥？** 世界上最受欢迎的程序员写字板。它就是你的主基地，所有的法术（代码）都在这里执行。
* **为啥要装？** 它是所有代码与 AI 插件运行的最佳舞台！没有它，就像画画没有画纸。
* **下载传送门：** [ 点击这里去官网免费下载 VSCode](https://code.visualstudio.com/) （下载后一路狂点下一步傻瓜式安装即可！）

### 2.  **OpenClaw**
* **这是啥？** 提供强大 AI 能力的开源引擎，就像是给你的程序装上一个超级涡轮。
* **为啥要装？** 它是让你的应用拥有大脑的底层核心服务，帮你流畅对接各种大模型。

### 3.  **Claude Code**
* **这是啥？** 驻扎在你电脑终端里的超级 AI 员工！
* **为啥要装？** 它可以直接操纵你的电脑帮你改代码、做设置。遇到报错不再需要自己去搜，只要安好它，你只需动动嘴皮子即可。

##  第二步：复制粘贴，召唤神龙（一键安装提示词）

准备好了吗？在你的 AI 对话框里，直接把下面的咒语（提示词）扔给它，然后舒舒服服喝口茶，看它自己操作！

###  1. 召唤 OpenClaw 引擎 
> **提示词：**
> AI 小助手你好！请帮我一键安装和配置 OpenClaw 环境。OpenClaw 的参考链接是：https://github.com/OpenClaw/OpenClaw。请你在终端中自动克隆代码、安装所需的依赖包，并进行必要的初始化配置。完成后，请帮我在本地启动服务，并告诉我如何用大白话验证它是否运行正常。

###  2. 召唤 Claude Code 助手
（请确保你已经打开了 VSCode，并且知道对话框在哪哦！）
> **提示词：**
> 我现在处于 VSCode 环境中，请在当前的集成终端里，帮我一键全局安装 Claude Code 命令行工具。安装完成后，请用幼儿园小朋友都能听懂的话告诉我，如何完成身份验证，并验证它是否已经满血复活可以干活了？

##  第三步：给龙虾喂糖果（获取 API Key 与配置聊天系统） 

光有龙虾 (OpenClaw)这个强力引擎还不够，它还需要吃一种叫 **Token (或者叫 API Key)** 的魔法糖果才能发动。有了它，你的龙虾就能连上最聪明的大脑（比如 Qwen 或 ChatGPT），并且为你建立一个专属的聊天工作台！

###  去哪找魔法糖果？
你需要去各大模型官网注册一个账号，获取并复制一把独一无二的秘钥（通常是一长串英文字母），请不要泄露给别人：
* **通义千问 (Qwen)：** 阿里云的大模型，对国内新手最友好，甚至还有免费的额度！去阿里云百炼平台注册，找到并创建一个API-KEY拿来用即可。
* **ChatGPT (OpenAI)：** 地表最强老大哥。你可以去 OpenAI 官网开发者平台获取，或者通过一些国内合法的第三方中转购买。

###  给龙虾建一个专属聊天室（聊天工作平台）
有了聪明的大脑，我们总不能对着干瘪的黑框框下指令吧？我们需要一个漂亮的网页或插件界面（比如 LobeChat 或 AI 插件）来跟龙虾对话。**别头疼该下什么软件，这依然可以交给 AI 助手帮你搞定！**

> **一键配置 Token 与搭建聊天室提示词：**
> AI 小助手你好！我已经拿到了大模型的魔法钥匙（API Key），它是：【把你复制的那一大串字母粘贴在这里，比如 sk-xxxx...】，我使用的是【通义千问 / ChatGPT 等】模型。
> 请帮我把这个秘钥注入并配置到 OpenClaw 服务或环境变量中。配置好后，请帮我在电脑上推荐并一键初始化一个好用的、带输入框气泡的聊天工作界面（比如 LobeChat 本地版或 ChatGPT Next Web），并且帮我把刚刚的秘钥绑定进这个聊天室里，让我可以直接在漂亮的网页里跟模型打字聊天！

##  第四步：Git  你的本地时光机 

写代码最怕什么？手滑删错了东西找不回来！Git 就是专门用来给你项目定时存档的时光机。**记住，你不需要学一行命令，全交给 AI 来办！**

###  一键基础配置提示词
> **提示词：**
> 请帮我检查一下我当前的本地 Git 状态是否正常。如果我还是个新手白板，请帮我设置好全局的 Git 用户名和邮箱（请主动问我想登记什么名字，并在我回答后帮我配置好）。

###  高频日常存档提示词
当你写完了一段代码，想存档或者上传到云端（GitHub）时，直接对 AI 说：
> **提示词（保存并上传）：**
> 我刚才完成了一些修改，请帮我把目前所有的改动进行暂存（相当于打包），并生成一个提交存档（简单写一句话说明我改了啥），最后帮我推送到云端仓库保存！

> **提示词（查状态）：**
> 请帮我看看我现在修改了哪些文件？还有没有没保存/没推送的改动？

##  终极奥义：遇到红字报错/小怪兽怎么办？ 

新手村最常见的怪物就是报错信息（终端里一段看起来很吓人的红色英文）。**千万别慌！别自己查词典！** 遇到问题，直接用下面这条咒语召唤你的 AI 导师：

###  自动除虫排障提示词
> **排障提示词：**
> 我在执行【这里写你在干嘛，比如：安装 Claude Code / 启动服务】时遇到了报错红字！以下是弹出的全部内容：
> 【请把红色的粗糙报错英文全部复制粘贴到这里】
> 不要让我去查文档！请你直接分析出错原因，**你有权限直接运行命令修复，请试着自己解决**。如果必须我手动操作（如关代理、重启电脑等），请分步骤用最傻瓜式的中文告诉我。

>  **龙师父的叮嘱：** 当你在 VSCode 里使用具备操作权限的智能体时，千万别把我们当只会陪聊的百科！告诉我们**自己连终端去改！**，我们能在你的文件夹里上天入地，直接帮你把 Bug 捏碎！
