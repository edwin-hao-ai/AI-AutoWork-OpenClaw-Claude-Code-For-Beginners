# OpenClaw 与 Claude Code 一键安装与配置指南

本文档提供了专门为具备自主操作能力的 AI 智能体调用的一键提示词。您可以直接复制以下提示词框内的内容发送给 AI，让它自动为您完成相关环境的安装与配置。

## 1. 一键安装 OpenClaw 提示词

用于向 AI 下达自动下载并配置 OpenClaw 服务的指令。

> **提示词：**
> 请帮我一键安装和配置 OpenClaw 环境。OpenClaw 的参考链接是：https://github.com/OpenClaw/OpenClaw。请你在终端中自动克隆代码、安装所需的依赖包，并进行必要的初始化配置。完成后，请帮我在本地启动服务，并告诉我如何验证它是否运行正常。

## 2. 一键安装 Claude Code 提示词

在您已经打开了 VSCode，并且具备 Node.js/npm 环境的情况下使用。

> **提示词：**
> 我现在处于 VSCode 环境中，请在当前的集成终端里，帮我一键全局安装 Claude Code 命令行工具（请执行 
pm install -g @anthropic-ai/claude-code 或最新推荐命令）。安装完成后，请告诉我如何通过 claude auth 等命令完成身份验证，并验证 CLI 工具是否已经可用。

## 3. 配置大模型 Token 与聊天工作台提示词

OpenClaw 需要配置大模型（如通义千问、ChatGPT等）的 API Key（Token）才能工作。您可以前往对应平台（如阿里云百炼平台或其他服务商）申请 API Key。

> **提示词：**
> 我已经获取了模型的 API Key：【请在此处粘贴您的 API Key】，我使用的是【通义千问 / ChatGPT 等】。请帮我将此秘钥配置到 OpenClaw 服务中。配置完成后，请帮我推荐并接入一个好用的带界面的交互聊天台（如 LobeChat 或其他相关页面/插件），让我可以直接通过页面对话。

## 4. 一键匹配本地 Git 提示词及使用说明

用于让 AI 检查您的环境并配置好 Git，同时提供基础的 Git 操作提示词，让你不需要记命令即可完成版本控制。

> **查看状态提示词：**
> 请帮我检查当前工作区，看看修改了哪些文件，是否有尚未提交的改动？
>
> **保存并推送提示词：**
> 我刚才完成了一些修改，请帮我将所有改动暂存、生成提交记录（写一句简短明了的修改说明），最后帮我推送到远程仓库保存。

**附：Git 常用命令备忘录（供手动操作时使用）**
- **查看状态**：git status
- **暂存改动**：git add .
- **提交记录**：git commit -m "更新说明"
- **推送云端**：git push

## 5. 遇到插件或不明问题的解决指南

如果在安装途中遇到网络中断、插件冲突或报错等任何状况，直接复制报错内容使用本提示词让 AI 协助。

> **排障提示词：**
> 我在执行【请替换为您正在进行的操作，例如：通过 npm 安装 Claude Code】时遇到了问题。以下是终端输出的完整报错：
> 【请在此处粘贴所有的报错文本信息】
> 请帮我分析出错的根本原因。你有权限直接运行命令修复或安装缺失的依赖，请试着自己解决。如果需要我手动配合操作（如关闭代理、重启等），请分步简单说明。

###  智能体协同建议
- **充分放权**：当你在 VSCode 中使用具有终端执行权限的 AI 时，直接告诉它**你有权限直接运行命令修复，请试着自己解决**，能大幅提升效率。
- **提供纯文本**：AI 对纯文本的错误日志解析最准，建议优先复制 Terminal（终端）里的报错文本，而不是发截图。
