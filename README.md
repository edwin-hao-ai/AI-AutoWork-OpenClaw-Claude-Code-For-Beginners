# 小白也能玩转 Claude Code + OpenClaw（龙虾）
## 让 AI 从聊天工具跨越到超级打工 OS演讲稿

本仓库是为面向零基础职场大众撰写的完整演讲/教学课件。总共设计为一场 90 分钟的课程。

###  2026年改版重大升级说明：
在保留了原先丰富的互动话术（PPT参考、互动扣1）以及生动的大白话类比基础上，深度整合了进阶系统观念，避免了早先章节中内容重合和逻辑跳跃。
*   **新引入概念**：软件时代变迁史（AI安装理念的颠覆）、操作系统定位。
*   **三大武器确立**：Claude Code（大脑）+ OpenClaw（双手）+ Git（降维的保险柜体系）。
*   **工作流效能**：文档工程化 Prompt 控制法；高价值心法：谋定而后动，先 Plan 后 Auto拒绝盲盒消耗。包含单点简单流与利用 Skill 构建复杂流的区别。
*   **底层原理解密**：剖析 Agent 基于文本的 Memory 记忆机制及其导致缓慢/耗费 Token 的瓶颈，引出 Awareness 前沿感知趋势。

> **演讲指导建议**：
> 不要陷入写代码或者配置环境的繁文缛节，核心是**建立认知**。当大众理解了不直接执行先看计划图、用文档代替闲聊写规则、拿Git当保险柜分享柜来用，这堂课的价值就彻底实现了。
