## 大白话术语速查表

> 演讲时如果忘了类比怎么说，翻这里查。

| 专业词 | 大白话类比 |
|--------|------------|
| Claude Code | AI 帮你"写代码+干活"的指挥台，你说需求它执行 |
| OpenClaw（龙虾）| AI 的"全自动跑腿小弟"，帮 AI 的指令落地变成真实操作 |
| GitHub | 版本管理+团队实时同步平台，龙虾可帮你全程配置，告别微信传文件 |
| AGENTS.md | 龙虾的 AI 员工手册，写一次规矩，龙虾永远照做 |
| CLAUDE.md | Claude Code 专属的员工手册，和 AGENTS.md 作用相同但属于不同工具 |
| SOUL.md / USER.md | 龙虾的性格文件和用户偏好文件，配合 AGENTS.md 使用 |
| 仓库（Repository） | 一个项目文件夹，放在 GitHub 的云上 |
| Chat 模式 | 你问我答的聊天方式，答完就忘 |
| Agent 模式 | 给 AI 定好规矩+工具，让它自动执行任务 |
| 自动化 | 以前要你手动做的事，现在 AI 全包了 |
| 提示词（Prompt）| 你告诉 AI 要干什么的那段话，越清楚越好 |
| 工作流（Workflow）| 把一件事拆成一步一步，AI 按顺序干完 |
