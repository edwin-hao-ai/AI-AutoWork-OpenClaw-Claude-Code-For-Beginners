## 四、Q&A 引导⏱ 约 3 分钟

> [PPT参考] 第27张幻灯片：Q&A 大字 + "有问题，举手"

【讲师提示】Q&A 环节把最常见的3个问题先自己说出来，打破观众的沉默，然后再开放提问。

**先说三个我猜你们最想问的问题：**

**Q1：Claude Code 要钱吗？**

Claude Code 本身免费，但使用 Claude AI 的次数是计费的，大概用法是按"你说了多少话、AI 回了多少话"来算。普通职场用途，一个月大概花几十块钱。

国内平替版（腾讯 WorkBuddy、Kimi Claw 等）有免费额度，建议先用免费版试手感，满意了再付费。

**Q2：我的数据安全吗？数据会不会被 AI 学走？**

这是个好问题。默认情况下，Claude 不会用你的数据训练模型（可以在账号设置里确认）。如果你处理的是特别敏感的数据，建议用国内平替的本地部署版本（阿里云 CoPaw、阿里 QoderWork），数据留在你自己的电脑上，完全不出去。

**Q3：我是苹果电脑用户 / 我是 Windows 用户，有区别吗？**

基本没有区别，今天讲的所有工具都支持 Mac 和 Windows。我在 GitHub 仓库里会放两个系统的分步截图指南，遇到问题翻仓库就行。

**好，还有其他问题的，现在可以在聊天框里打出来，或者直接开麦问我。**
